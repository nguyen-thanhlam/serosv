# 📌 Research Overview
This repository serves as a showcase of work completed during my internship with the Mathematical Modelling Group at the Oxford University Clinical Research Unit (OUCRU). The project focuses on *serosv*, an R package designed to estimate critical epidemiological parameters - such as the Force of Infection (FoI) and Basic Reproduction Number ($R_0$) - from serological data.

My work involved implementing and validating statistical models to better understand the transmission dynamics of infectious diseases using cross-sectional seroprevalence datasets.
